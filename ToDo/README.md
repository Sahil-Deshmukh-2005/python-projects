# ToDo List

A simple File Handling project that keeps track of your tasks.

***
## Functions in ToDo 
### 1.Add a task.
### 2.Remove a task.
### 3.Update a task.
### 4.View all tasks.
### 5.Number of tasks remaining.
### 6.Delete all tasks including the file.

***

## Concepts used
File handling , is used to keep the track of all the tasks.  
Funtions , to keep the code clean.  
OS Module , To check if the file exist or not and to delete it , included the `os` library.
